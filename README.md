# AI-Powered ToDo Helper

## Description
A simple ToDo application enhanced with AI. Users can add tasks and the AI improves the task descriptions, making them clearer and more actionable.

## Features
- Add tasks
- AI-powered task text improvement
- Dynamic task list display

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenAI GPT API

## How to Use
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter a task in the input field and click "Add Task".
4. The AI will optimize the task description and display it in the list.

## Future Improvements
- Mark tasks as completed
